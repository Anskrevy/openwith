Install:

Open With needs a file outside of the browser to communicate with.

  1. You'll need to have Python installed for this to work. Fetch it from here.
  2. Click here to download Click here to download Click here to download and save the file to your computer.
  3. Ensure the file permissions allow you to run it ("x" permission): 
    chmod u+x open_with_linux.py
    chmod u+x open_with_mac.py
  4. Open a command prompt. Run the file with the argument "install", like this: 
    open_with_linux.py install
    open_with_mac.py install
    open_with_windows.py install
    If you move the file, you must run this step again.
